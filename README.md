## Getting Started

### Installation

1. Clone the repo

```sh
git clone https://github.com/BubbleKID/dynamic-table.git
```

2. Install NPM dependency:

```sh
npm install
```

3. Start

```sh
npm run dev
```

### Testing

This project use `Jest/Enzyme` framwork to create the unit testing.

```sh
npm test
```

### Build

This project build with webpack4.

```sh
npm run build
```
